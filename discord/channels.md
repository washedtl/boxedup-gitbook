# Channel map

🟢 **Beginner.** A guide to the BoxedUp Discord channel structure — what each channel is for, how to use it, and what NOT to post where.

> **Note:** This page is a starter — replace channel names with the actual ones in your server. The structure described below is a common BoxedUp-style layout.

## Top-level structure

The Discord is organized into 5 categories:

```
📌 — START HERE
💬 — COMMUNITY
🎯 — DEAL FEEDS (live alerts)
🎓 — LEARNING
🛠️ — TOOLS & INFRASTRUCTURE
```

Each category has channels with specific purposes. Mixing them slows everyone down.

## 📌 START HERE

### `#welcome`
First channel new members see. Read the rules. React to the role-assignment message to get access to the rest.

### `#rules`
The non-negotiable rules:
1. No drama, no doxing, no harassment
2. No referral links unless explicitly approved
3. No selling products to other members (use the marketplace channel if it exists)
4. Don't repost deals from paid groups — respect the source
5. Be respectful of newer members — we all started somewhere

### `#announcements`
BoxedUp-wide announcements. Big strategy shifts, big restocks, server-level news. Read-only.

### `#guide-feedback`
Where to report errors or suggestions for this GitBook.

## 💬 COMMUNITY

### `#general`
Casual chat. Memes, business banter, off-topic-ish but reseller-adjacent. Don't post deals here (use deal channels).

### `#wins-and-losses`
Share your screenshots: \$500 day, sold-out replens, also painful losses. Vulnerability welcome.

### `#introductions`
New members say hi. Helpful to mention: where you're from, what you sell, what stage you're at. Veterans use this to identify who to mentor.

### `#sourcing-help`
Stuck on whether to buy something? Post the ASIN, your numbers, your concern. Veterans will give read on it. **Not** a place to ask "is this a good deal" without doing your own analysis first.

### `#account-issues`
Suspension help, IP complaint help, customer-service shockers. Detailed accounts of issues + community-sourced solutions.

## 🎯 DEAL FEEDS

This is what BoxedUp is famous for. Multiple channels stream live deals.

### `#a2a-feed` (Amazon-to-Amazon)
Real-time A2A picks — Amazon items currently mispriced. See [A2A strategy](../sourcing/a2a.md). Acted on in minutes.

### `#restock-monitors`
Tempo and Blaze pings when monitored ASINs come back in stock. See [Restock monitoring](../sourcing/restock-monitoring.md).

### `#kpf-queries`
Members share KPF queries that worked. Copy → adapt → run.

### `#walmart-clearance`
Walmart in-store + online clearance leads.

### `#target-clearance`
Target Redsky-discovered clearance.

### `#wholesale-leads`
Verified wholesale opportunities. Non-public, often gated to higher tiers.

> **The deal-feed protocol**: when you act on a deal someone else surfaced, react with ✅ or 💰. Helps the source know their leads convert. Also: don't double-post the same deal across channels.

## 🎓 LEARNING

### `#guide`
Pinned link to this GitBook. New members start here.

### `#case-studies`
Members write up their wins/losses in detail. The RAM shortage case study, big restock plays, sourcing methodology breakdowns. Long-form, high-signal.

### `#resources`
Tool reviews, prep service recommendations, accountant referrals, software discounts. Curated.

### `#q-and-a`
Numbered, async question-and-answer threads. "How do I unstick a stuck shipment?" type questions get answers from veterans + a permanent record.

## 🛠️ TOOLS & INFRASTRUCTURE

### `#tempo-status`
Health checks for the Tempo monitor. Outage notifications.

### `#blaze-status`
Same for Blaze.

### `#bot-commands`
Where you run bot commands without spamming public channels. Tempo `!m add`, etc.

### `#tool-bug-reports`
Found a bug in BoxedUp Command Center, the Discord bots, or other tooling? Post here.

## What NOT to post (and where)

| Bad | Better |
|---|---|
| "Anyone selling on Amazon?" | `#introductions` with details |
| Random deal screenshot in `#general` | Specific deal channel |
| "Is this a good deal?" without numbers | Fill in numbers, post `#sourcing-help` |
| "My account is suspended help!!!" | `#account-issues` with details |
| Promoting your service/discord | DM mods first; usually disallowed |
| Reposting another paid group's leads | Banned — gets you removed |

## Channel etiquette

Five rules that keep the server scaling:

### 1. Use threads for long discussions
A 50-message back-and-forth in `#general` clutters everyone's notifications. Right-click the original message → Create Thread → continue there.

### 2. React, don't type "+1"
A 👍 reaction = 1 ping for the message author. Five "+1" replies = 5 pings.

### 3. Drop the ASIN, then the numbers, then the question
A useful sourcing-help post: `B0XXXXX`, "Cost \$8 RA, Buy Box \$25, BSR 8K, 2 FBA sellers, [Keepa link]. Worried about offer count spike — thoughts?"

A useless post: "Should I buy this?" + a Walmart screenshot

### 4. Search before asking
Most beginner questions have been asked 50 times. Use the magnifying glass. Read the GitBook.

### 5. Don't ping moderators for non-emergencies
Mods are members too. `@here` and `@everyone` are silent — if you accidentally trigger them, you'll catch a moderation flag.
